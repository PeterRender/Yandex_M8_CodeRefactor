# Анализ результатов рефакторинга

## 1. AddressSanitizer (leak_example.cpp)

**До рефакторинга:**
ERROR: AddressSanitizer: new-delete-type-mismatch
allocated type: 8 bytes (Derived)
deallocated type: 1 bytes (Base)

**После рефакторинга:**
- Ошибок нет, память освобождается корректно

**Вывод:** ASan обнаружил утечку, утилита ее исправила добавлением `virtual`.

## 2. Замер времени выполнения (perf_example.cpp)

| Метрика | До (копирование) | После (ссылка) | Изменение |
|---------|------------------|----------------|-----------|
| real    | 2.737s           | 2.627s         | -4%       |
| user    | 0.641s           | 0.549s         | **-14%**  |
| sys     | 2.094s           | 2.074s         | -1%       |

**Вывод:** Уменьшение user time на 14% после того, как утилита заменила копирование на ссылку (с учетом оптимизации -O2)
